## About The Project

ISRO Rocket Mission Simulator is a Python-based interactive space exploration game inspired by real-world space missions. Players take the role of an ISRO scientist, choose a destination in the Solar System, name their rocket, schedule a launch, and face random challenges during the mission.

The simulator includes realistic concepts such as weather delays, space hazards, travel times, and mission reports based on the selected destination.

## Features

* 🌕 Explore the Moon and all major planets
* 🚀 Name your own rocket
* 📅 Schedule launch dates
* 🌦️ Random weather conditions and launch delays
* ☄️ Random space events and mission failures
* 🛰️ Mission reports for each destination
* ⌨️ Typewriter-style dialogue effects
* 🎲 Randomized gameplay for replayability

1. Install Python 3.
2. Install the required package:

```bash
pip install python-dateutil
```

3. Run the program:

```bash
python main.py
```
